NSW Education Kookaberry Scripts
--------------------------------

A repository for MicroPython scripts provided by the AustSTEM Foundation for use by the NSW Department of Education in the TAS curriculum.

This repository is experimental and is subject to a lot of changes.

This repository is organised into folders:
* apps - MicroPython source code
* lib - Library modules, usually bytecode MPY files
